RoboNope Static Content Example

About This Page

This is an example of static content that can be served by the RoboNope module when a bot ignores robots.txt rules.

Instead of generating dynamic content, you can configure the module to serve pre-made static pages like this one.

Features

  • Customizable static content
  • Consistent responses for bots
  • Lower CPU usage compared to dynamic content
  • Can include honeypot links
